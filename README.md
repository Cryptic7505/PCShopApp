# PCShop App
A Java application designed to streamline the PC building and parts purchasing process for customers, minimizing the need for staff assistance. It features a database-driven interface for displaying and selecting components. The app will automatically generate a build receipt and submit the order to the shop's database, with immediate stock deduction or order queuing. Furthermore, the system includes comprehensive inventory and order management functionalities for the shop, such as adding new stock, registering new parts with their specific identifiers, and removing existing parts from the inventory/database. <br><br>
The application has the following functionality:
* Part picker interface (no part compatibility checks)
* Core Components and Peripherals selection
* Name,Contact, and address information storing to database
* Order/Build list stored in the database with date
* Shop Owner/Employee UI for Data Display and Stock management
